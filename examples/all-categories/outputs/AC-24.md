# LocalNote 1.0 安装指南

本指南旨在引导 Windows 用户完成 LocalNote v1.0 的安装与基本验证，确保软件顺利运行。

### 一、 安装步骤
1. **启动安装**：在 Windows 文件资源管理器中定位至下载的安装包文件。
2. **执行安装**：双击安装包图标，按照屏幕提示完成安装流程。
3. **启动程序**：安装完成后，双击桌面或开始菜单中的 LocalNote 快捷方式运行软件。

### 二、 安装验证
安装完成后，请通过以下步骤确认软件版本及状态：
1. 查看软件主界面标题栏，确认显示的版本号为 v1.0。
2. 在编辑器内输入测试文字，确认软件响应正常且功能可用。

### 三、 卸载指南
如需移除 LocalNote，请按照 Windows 标准程序管理流程操作：
1. 点击“开始”菜单 $\rightarrow$ “设置” $\rightarrow$ “应用” $\rightarrow$ “安装的应用”（或“程序和功能”）。
2. 在列表中找到 LocalNote。
3. 选择“卸载”并确认，等待系统完成清理工作。

### 四、 注意事项
- 确保安装路径所在磁盘有足够的可用空间。
- 建议在安装完成后重启计算机以保证软件环境稳定。
